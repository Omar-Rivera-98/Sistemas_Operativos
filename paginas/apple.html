Apple

La primera versión de macOS para el escritorio, Mac OS X 10.0, fue lanzada por Apple en marzo del 2001, marcando un comienzo en la revolución del sistema operativo como tal y facilitando al usuario el uso de la computadora.

¿De quién fue la idea?

jef Raskin, quien comenzó el proyecto Macintosh en 1979, sin olvidar que Steve Jobs era el líder que jugó un papel importante en la creación de este sistema operativo.

Proceso de la elaboración del sistema operativo:

El procesos que tomó este sistema operativo fue algo clave para que fuese innovador, se opto por una interfaz gráfica de usuario (GUI) innovadora, sirvió como base para el desarrollo de nuevas versiones de MacOS, implementando los iconos, el uso del mouse, se basa en UNIX, permitiendo potencia y flexibilidad.

Idea revolucionaria

Mac Os en sí mismo es una familia de sistemas que han evolucionado a través del tiempo con diferentes versiones, su principal característica es que está diseñado para funcionar de manera optimizada en equipos Apple ofreciendo un entorno virtual en forma de “escritorio” que facilita la interacción entre el computador y el usuario, fue un sistema que innovó en mucho apartados, como el uso de ventanas, el uso del mouse, un sistema rápido para el hardware diseñado (por apple por supuesto).

Actualidad

MacOS ha evolucionado bastante, adaptándose a las necesidades actuales como la interfaz moderna, incluyendo un dock que es para anclar los programas más utilizados, un finder para buscar programas y archivos, un escritorio para tener acceso a los programas de una manera más fácil.
